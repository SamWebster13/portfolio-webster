Description

This is a image generator, thats trying to replicate the images in the drop down. It randomly creates and draws a set number of images, using color sampling to match up colors, and aranges them to better remake the image.

Technical

Essentially the generator runs mutations on the shapes and colors, and is given a score based on how close it is to the original image. It remembers this score and what it generated to get there, so the algorithm learns and goes from there, slowly getting closer to the original as it continues to mutate
