Experiment 4 - Infinite Worlds

Description

Im calling this my infinite variable miniverse generator. Each dot is supposed to represent a small solar system, the arrow orbiting it the user's space ship. Dashed lines connect the ship's orbit to other near by stars that are in range for the ship to jump to.

Technical

These are made through the repeated use of several main functions, mainly a repetitive draw function, a drawTile function, drawShip, and drawDashedLines (for the hyperlink lines). These functions draw and create the universe, animate the ships orbit, and draw the lines that connect 'in-range' star systems.

Reflection

While working on this project I was constantly seeing flashes of a game called FTL, where you pilot a starship racing against seditionist rebels to get a message to 'Federation HQ.' One of the mechanics involved choosing which star system / star cluster you went to next, and it all looked fairly similar to this generation.
