Experiment 1 - Generating Text / Stories

Description

This Experiment was done to try and make a story engine, that would randomly make a basic plot for a story,keeping genre in mind. For example, while each run would be a random roll of genres, subgenres, and characters,

I made it so that you wouldnt find a wizard in a sci fi western shootout, because while that may be interesting, it'd certainly be hard to write.

Technical

Basically I made several groupings based on genres, so each genre, sci fi, fantasy, horror, mystery etc, would have a list of different subgenres, characters, moods, and other descriptors to make story generation more specific.

When the generator picked a general genre at random, it would be filtered into these lists. If the generator picked the mystery genre, some sub genres would be "Noir", "Supernatural", "Historical", and some characters would be detective, librarian, or reporter, etc.
