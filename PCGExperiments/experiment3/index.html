Experiment 3 - Alternate Worlds

Description

This Generator uses a tilesheet to randomly render miniature landscapes, mapping the different kinds of tiles with an ascii table, which can be edited and interacted with in real time.

Technical

These landscapes are created through several algorithms, first generating a grid from a seed, and then drawing, placing tiles from the asset sheet into the keyed areas on the grid.
